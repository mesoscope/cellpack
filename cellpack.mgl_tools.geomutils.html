cellpack.mgl_tools.geomutils.surface.findComponents(verts, faces, normals=None, returnOption=0)[source]¶

find the components of a geometry. normals are normals of verts not faces. returnOptiont: component return option

= 0: return all components = 1: return all outside surfaces (volume > 0; normals must be given)

The code is based on the Vision node ConnectedComponents.
